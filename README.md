## 📌 О проекте

**RandomatusBot** — это бот для проведения конкурсов, розыгрышей и гивов (giveaways) в Telegram на базе **Telegram Mini Apps**. Бот поддерживает как классические механики выбора победителей, так и инновационные механики.

<br>

## 🎯 Ключевые возможности

| Возможность | Описание |
|---|---|
| 🎲 **Честный рандом** | Победители определяются криптографически случайным алгоритмом. |
| ✅ **Проверка подписок** | Автоматическая проверка условий участия (один или несколько каналов/групп) через Telegram API — в момент участия и повторно перед итогами |
| 🕹 **Геймификация (слот-машина)** | Добавлен новый интерактивный формат: участник крутит слот и мгновенно узнаёт результат |
| ⭐ **Монетизация через Stars** | Продажа дополнительных билетов/спинов участникам с выводом средств организатору |
| 🛡 **Антифрод-защита** | Выявление фейковых аккаунтов, ботоферм и накруток с автоматическим исключением из пула |

## 🚀 Быстрый старт: конкурс за 3 минуты

Розыгрыш полностью настраивается внутри мессенджера.

**Пошагово:**

1. **Добавление бота** — добавьте [@RandomatusBot](https://t.me/RandomatusBot) в канал или группу с правом публикации сообщений.
2. **Запуск бота** — Нажмите кнопку "Открыть", чтобы запустить веб-интерфейс бота.
3. **Создание конкурса** — через пошаговое меню:
   - выберите формат конкурса
   - выберите формат завершения: по дате/времени или по достижении нужного числа участников
   - введите текст конкурсного поста, добавьте изображение, настройте кнопку поста
   - введите количество призовых мест и способы получить доп. билеты\спины за активность ( за голос каналу или за Telegram Stars. Реферальная система включена по умолчанию )
   - выберите каналы для обязательной подписки.
4. **Старт** — бот публикует пост и дальше работает в автопилоте: сбор, проверка подписок, выбор победителей, уведомления.

## 💼 White Label: брендированные боты

Для каждого пользователя доступна опция создания персонального бота для конкурсов под собственным брендом.

- 🏷 **100% свой бренд** — название, аватар, описание и логика бота полностью ваши, без упоминаний RandomatusBot.
- 📊 **Лидогенерация** — каждый участник, открывший бота, навсегда сохраняется в вашей централизованной базе.
- 📢 **Рассылки** — встроенные массовые рассылки (посты, медиа, акции) по всей накопленной базе без ограничений.
- 💰 **Прямой заработок** — платежи в Telegram Stars за дополнительные шансы поступают напрямую на ваш баланс со 100% выводом.

## ❓ FAQ

<details>
<summary><b>Что такое брендированный бот для розыгрышей?</b></summary><br>

Брендированный бот — это развёртывание RandomatusBot в собственном боте пользователя. Все участники, запустившие бота, попадают в вашу собственную базу. Вы можете делать по ним рассылки и выгружать данные для своих маркетинговых целей в любой момент.
</details>

<details>
<summary><b>Чем слот-розыгрыш отличается от классического конкурса?</b></summary><br>

В классическом конкурсе участники собираются до дедлайна, после чего выбирается победитель. В слот-машине пользователь совершает игровое действие сразу же. Призовой фонд строго фиксирован организатором заранее, что исключает казино-механику и превращает процесс в интерактивный маркетинговый инструмент.
</details>

<details>
<summary><b>Безопасна ли покупка дополнительных билетов за Stars?</b></summary><br>

Да, все транзакции обрабатываются через официальный платёжный шлюз Telegram Stars. Покупка билетов увеличивает долю пользователя в общем пуле розыгрыша, но финальный выбор победителя остаётся за независимым криптографическим алгоритмом.
</details>
